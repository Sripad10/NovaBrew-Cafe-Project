# NovaBrew Coffee Taste Profile Quiz

A high-fashion, editorial coffee personality quiz built for NovaBrew Coffee Co. The experience helps subscribers discover their coffee profile and pairs them with curated NovaBrew recommendations.

## Live Concept

The quiz is designed to solve NovaBrew's retention problem by making subscribers feel known instead of randomly matched. It combines:

- abstract and visual personality questions
- percentage-based results
- signature coffee recommendations
- a luxury European editorial design direction

## Coffee Personalities

- **Bold Explorer**: intense, expressive, and drawn to stronger flavor profiles
- **Smooth Operator**: balanced, polished, and effortlessly approachable
- **Cozy Classic**: soft, comforting, and ritual-driven
- **Wild Card**: adventurous, curious, and energized by surprise

## Built With

- Next.js
- React
- Tailwind CSS
- Static export for GitHub Pages deployment

## Local Development

Run the development server:

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Deployment

This project is configured for static deployment on GitHub Pages.

## About

Built as a customer personalization prototype to improve subscriber retention through better taste matching, stronger identity, and a more premium onboarding experience.

## Why It Matters

Instead of a generic subscription quiz, this prototype positions NovaBrew as a brand that understands subscriber taste from day one and can build a longer-term personalization loop around it.
